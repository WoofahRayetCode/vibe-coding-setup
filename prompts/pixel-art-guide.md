# 🎨 2D Side-Scrolling Pixel Art Prompt Guide

Use the following highly engineered prompt blueprints and terms to generate flawless, premium-quality assets using the `vibe-asset` local tool. 

---

## 1. Character & Enemy Sprites (Side-View)
When generating characters or monsters for side-scrollers, you need clear profiles, no dynamic perspectives, and solid flat backgrounds for clean transparency cropping.

### 📋 Blueprint Formula:
> `[Subject Description] in side-view profile, retro pixel art, 16-bit sprite, clean outline, flat solid [Color] background, isolated game asset, sharp detail`

### 💡 Example Prompts:
* **The Hero**: `retro knight character in metal armor, side-view profile, walking action pose, retro pixel art, 16-bit sprite, flat solid blue background, isolated game asset`
* **Goomba-style Enemy**: `cute purple slime monster with angry eyes, side-view profile, retro pixel art, 16-bit sprite, flat solid yellow background, isolated game asset`
* **Boss Monster**: `glowing mechanical robotic spider, side-view profile, retro pixel art, 16-bit sprite, flat solid green background, isolated game asset`

---

## 2. Tileable Platforms & Ground Blocks
For seamless terrain blocks, walls, or platforms, you must invoke the **tileable texture detection** built into the engine by using terms like "tile", "ground", or "seamless".

### 📋 Blueprint Formula:
> `[Terrain Material] block, retro pixel art, 16-bit, seamless tileable pattern, full frame texture, clean pixels`

### 💡 Example Prompts:
* **Classic Grass Block**: `mossy green grass top on dark brown dirt soil block, retro pixel art, 16-bit, seamless tileable pattern, clean pixels`
* **Sci-Fi Platform**: `metallic futuristic steel grid plating, retro pixel art, 16-bit, seamless tileable pattern, full frame texture`
* **Lava Cave Wall**: `cracked black obsidian rock wall with glowing lava veins, retro pixel art, 16-bit, seamless tileable pattern, clean pixels`

---

## 3. Parallax Background Layers
For rich 2D environments, generate flat, low-contrast layers that can be split and scrolled at different speeds in Godot or GameMaker.

### 📋 Blueprint Formula:
> `[Layer Description], flat 2D silhouette, retro pixel art, 16-bit, horizontal layer, solid [Color] sky background`

### 💡 Example Prompts:
* **Distant Layer (Far)**: `distant silhouetted sharp mountains, flat 2D silhouette, retro pixel art, 16-bit, solid light blue sky background`
* **Mid-ground Layer**: `misty pine forest trees silhouettes, flat 2D silhouette, retro pixel art, 16-bit, solid pale teal background`
* **Foreground Decor**: `scattered old stone ruins and columns, flat 2D silhouette, retro pixel art, 16-bit, solid bright green background`

---

## 4. Collectibles & UI Items
Items, power-ups, coins, and keys require extreme pixel readability at small scales.

### 📋 Blueprint Formula:
> `[Item Description], retro pixel art, 16-bit icon, flat solid [Color] background, high contrast, clean outline`

### 💡 Example Prompts:
* **Power-up**: `glowing crystal heart with a golden halo, retro pixel art, 16-bit icon, flat solid black background`
* **Key**: `rusty ancient iron skeleton key, retro pixel art, 16-bit icon, flat solid yellow background`
* **Weapon**: `broadsword with a blue sapphire embedded in the hilt, retro pixel art, 16-bit icon, flat solid black background`
